## Install YouTube as PWA
You will need a browser that supports PWAs. Most non-webview Chromium browsers support it.

1. Go to YouTube website ([youtube.com](https://youtube.com))
2. Enable desktop mode
3. Install site as an app
4. Disable desktop mode (optional)
5. Check your home screen for YouTube PWA
